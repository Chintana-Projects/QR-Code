# QR-Code-Generator
# (sem-2)


 ~ Executed using PyCharm ~
  
A Python-based QR code generator with customizable shapes, colors, and data types, supporting text, URL, phone number, and email address, with features including square or circular shapes, adjustable size, and customizable fill and background colors.

Features :

1. The user will be able to choose the kind of QR code to be generated.
   
          a) Text : when qr is scanned, the text will appear.
   
          b) URL  : when qr is scanned, particular website/youtube vedio will be opened.
   
          c) Phone Number : when qr is scanned, phone app will be automatically opened with the number already typed.
   
          d) Email :when qr is scanned, gmail app will be automatically opened, with the gmail id of the recepient already typed in.
   
2. The user can give a file location to which the qr code will be saved to.
   
3. The user will also be able to select the background color,fill color,size(5-30) and the shape of the qr inside(dotted / square).

NOTE: If the user enters a invalid shape/size/color then default options will be considered.
